# Lens.xyz Prototype

This is an attempt to recreate the UI [lens.xyz](https://lens.xyz)

## Stack

- TypeScript
- React
- TailwindCSS
- Framer motion

## To view this on your computer

```bash
git clone https://github.com/gzkdev/lens-xyz-prototype.git
```

```bash
#npm
npm install

#yarn
yarn add

#pnpm
pnpm install
```

```bash
#npm
npm run dev
```
